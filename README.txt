
SIMULATION OF SCHEDULER PORTION OF MEMORY CONTROLLER FOLLOWING CLOSED PAGE POLICY

Steps to follow to run the project-
Step 1: Run the python file- Team7_msd_project
Step 2: You can choose Enable or Disable Debug Mode. If debug is 'off', default trace file is processed.
Step 3: Enter the path of the trace file which contains CPU Requests with format: [time] [Core] [operation] [hexadecimal address]
Step 4: The output text file with a trace of all DRAM commands issues is generated
